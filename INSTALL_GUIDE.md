# 本地环境配置指南

检测到您的电脑尚未安装 **Node.js**，导致无法运行 `npm` 命令。请按照以下步骤快速完成配置。

## 第一步：安装 Node.js

这是运行本项目的必须环境。

1. **点击下载**：[Node.js 官网 (LTS 版本)](https://nodejs.org/zh-cn/)
2. **安装**：下载后双击安装包，一路点击“下一步”直到完成。
3. **验证**：安装完成后，**必须关闭并重新打开**您的终端（Terminal）或 Cursor 编辑器，然后输入以下命令检查：
   ```bash
   node -v
   npm -v
   ```
   如果显示版本号（例如 `v20.x.x`），说明安装成功。

## 第二步：启动项目

环境准备好后，在终端中依次运行以下命令：

1. **安装项目依赖**：
   ```bash
   npm install
   ```
   *(这一步会下载 React、Tailwind CSS 等所有需要的工具)*

2. **配置 API Key**：
   确保根目录下的 `.env` 文件里填好了你的 `DEEPSEEK_API_KEY`。

3. **运行网页**：
   ```bash
   npm run dev
   ```

4. **打开浏览器**：
   终端会显示一个链接（如 `http://localhost:5173`），按住 `Cmd` 点击该链接即可看到网页。
